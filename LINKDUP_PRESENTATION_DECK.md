## GLOBAL DESIGN SYSTEM

**Canvas Size:** 16:9 Widescreen (1920 × 1080px recommended export)

**Background Palette:**
- Primary Background: `#0F0F10` (near-black, warm undertone)
- Secondary Surface: `#161618` (card/panel backgrounds)
- Tertiary Surface: `#1E1E22` (subtle separation)

**Accent System:**
- Chrome Silver: `#C0C0C8` — used for rule lines, micro-labels, secondary body text
- Electric Blue: `#3D9BFF` — used sparingly for highlight moments, glow effects, key data points
- White: `#FFFFFF` — primary headline color
- Soft Off-White: `#F0F0F0` — body copy, supporting text

**Typography Stack:**
- Display Headlines: **Bebas Neue** — 96–180pt, tracking 0–20, all caps
- Sub-Headlines / Callouts: **Anton** — 48–72pt, all caps, white
- Section Labels / Eyebrow Text: **Neue Haas Grotesk Display** — 12–16pt, all caps, letter-spacing 4–6pt, chrome silver
- Body Copy: **Inter** — 16–24pt, regular/medium weight, off-white
- Supporting Detail: **Montserrat** — 12–14pt, medium, chrome silver

**Layout Grid:** 12-column grid. Safe margin: 80px left/right, 60px top/bottom.

**Texture Layer:** A very subtle film grain overlay at 4–6% opacity sits on top of every dark slide. This adds cinematic depth without being distracting.

**Lighting/Glow Effect:** Select slides use a radial gradient glow — soft electric blue at 8–15% opacity — centered behind key text or imagery. This creates a halo/stage-light feeling.

**Rule Lines:** 1px horizontal rules in chrome silver at 30% opacity used to separate sections within a slide.

**Photo Treatment:** All photography is treated with a high-contrast, slightly desaturated grade. A subtle dark vignette frames every photo. No white or light backgrounds in photos — source images should be moody, golden-hour, indoor ambient-lit, or studio-quality.

**Animation Default (all slides unless noted):**
- Transition IN: Fade 0.4s
- Headline: Appear + slight upward motion (8px), 0.3s ease-out, 0.1s delay after transition
- Sub-elements: Staggered appear, 0.15s between each

## PRODUCTION NOTES FOR DESIGNER

### File Setup
- Create in PowerPoint (.pptx) format, 16:9 widescreen, 1920×1080px.
- Alternatively build in Keynote or Google Slides for easier presenting, then export to .pptx.
- Set all transitions to "fade" (0.4s standard, 0.6–0.8s for featured slides).
- Use "Appear" animations with slight directional offset (upward 8px for text elements).
- Group animation triggers: all items on a slide should auto-advance on a timed sequence, not require individual clicks — except the main presentation slides which should advance on presenter click.

### Font Sourcing
- **Bebas Neue** — Google Fonts (free, embed in file)
- **Anton** — Google Fonts (free)
- **Neue Haas Grotesk Display** — Commercial (Monotype). Fallback: **Aktiv Grotesk** or **DM Sans** if budget is constrained.
- **Inter** — Google Fonts (free)
- **Montserrat** — Google Fonts (free)
- Embed all fonts in the final PowerPoint file.

### Photography Sourcing
- Primary recommendation: Shoot event photography in advance at The Ring Workspace and use real photos.
- Secondary: Source from **Unsplash** (unsplash.com) or **Pexels** — search terms: "creative networking event," "content creator studio," "photographer candid," "startup mixer," "creator community," "young professionals workspace," "Tampa Bay."
- All photos should be treated with the same photo grade: Lightroom preset or Photoshop adjustment — shadows lifted slightly, highlights pulled down, slight desaturation (–15), contrast +25, clarity +10, slight warm tone (+8 temperature).

### QR Code Generation
- Use **QR Code Generator** (qr-code-generator.com) or **Canva QR** to generate all QR codes.
- Export QR codes as SVG or high-resolution PNG (minimum 800×800px) for sharp reproduction on screen.
- QR codes must be tested on a phone before the event to verify all links are live and correct.

### Presenting Tips
- Use a wireless presentation clicker (Logitech Spotlight recommended — has a laser pointer and timer).
- Presenter view: enable on your laptop so you see notes and next slide while the audience sees the current slide.
- Set screen resolution to match your projector or display (1920×1080 preferred).
- Test all QR codes from audience distance before the event.
- Have a PDF backup of all slides in case PowerPoint fails.
- Music recommendation: Create a Spotify playlist with moody, ambient, modern instrumental tracks — think: Frank Ocean instrumentals, Kaytranada, Sango, or similar. Fade music to 20% volume during speaking portions.
